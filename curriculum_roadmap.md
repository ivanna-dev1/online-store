# План навчання: Проект "Онлайн Магазин"

Цей проект розроблений для закріплення навичок React (Props, State, Lifting State Up) та вивчення нових концепцій (Basic Routing, Business Logic in Frontend).

## Модуль 1: Фундамент та Каталог
**Мета:** Налаштування середовища та рендеринг списків.
- Ініціалізація проекту (Vite + React).
- Підключення TailwindCSS.
- Створення структури даних (масив товарів).
- Компонент `ProductCard` (відображення одного товару).
- Компонент `Catalog` (рендеринг списку через `map`).

## Модуль 2: Інтерактивність (Пошук та Фільтри)
**Мета:** Робота з контрольованими формами та похідним станом (Derived State).
- Створення пошукового інпуту (контрольований елемент).
- Фільтрація списку товарів за пошуковим запитом.
- Створення кнопок категорій.
- Множинна фільтрація (пошук + категорія).
- *Концепція:* Чому ми фільтруємо дані під час рендеру, а не видаляємо їх зі стейту.

## Модуль 3: Кошик (Lifting State Up)
**Мета:** Розуміння передачі даних між компонентами та управління спільним станом.
- Підняття стейту (Де зберігати кошик? -> `App` компонент).
- Функція додавання товару в кошик.
- Відображення лічильника товарів у Header.
- Логіка "Товар вже у кошику" (зміна кнопки на картці).

## Модуль 4: Навігація (React Router)
**Мета:** Створення багатосторінкового додатку (SPA).
- Встановлення `react-router-dom`.
- Налаштування маршрутів (`/` - головна, `/cart` - кошик).
- Використання `Link` для навігації без перезавантаження.
- Рефакторинг структури: розділення на `Pages` та `Components`.

## Модуль 5: Сторінка Кошика
**Мета:** Робота зі складними масивами та підрахунками.
- Відображення списку обраних товарів.
- Функція видалення товару з кошика.
- Підрахунок загальної суми (reduce).
- (Опціонально) Зміна кількості одного товару (+/-).

## Модуль 6: Фіналізація
**Мета:** Полірування UI та код-стайл.
- Винесення констант та даних у окремі файли.
- Перевірка UX (повідомлення, якщо нічого не знайдено).
- Підсумковий рев'ю коду.
