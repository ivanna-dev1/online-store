1_ОСНОВНИЙ ФУНКЦІОНАЛ

# 🟦 БЛОК 1: ТЕ ЩО МИ ВЖЕ ЗРОБИЛИ (Completed)

Цей розділ описує кроки, які ми вже пройшли. Використовуй цей список для відтворення проекту або перевірки логіки.

### 📦 Модуль 1-4: Базовий функціонал

- **Крок 1:** Ініціалізація проекту за допомогою Vite (`npm create vite@latest`) та очищення стандартних стилів.
- **Крок 2:** Встановлення та налаштування TailwindCSS v4 для сучасної стилізації.
- **Крок 3:** Створення бази даних у файлі `src/products.js` (масив об'єктів з id, name, price, category, image, description).
- **Крок 4:** Створення компонента `ProductCard`, який приймає дані через `props` та відображає картку товару зі стилями Tailwind.
- **Крок 5:** Створення `CatalogPage` для відображення списку всіх товарів через метод `.map()`.

### 🔍 Модуль 2: Інтерактивність та Пошук

- **Крок 6:** Додавання стану `searchTerm` (`useState`) для контрольованого інпуту пошуку.
- **Крок 7:** Реалізація динамічного списку категорій за допомогою `new Set()` з масиву товарів.
- **Крок 8:** Додавання стану `selectedCategory` для фільтрації товарів за категоріями.
- **Крок 9:** Реалізація комбінованої фільтрації: список товарів оновлюється одночасно враховуючи і пошуковий запит, і вибрану категорію.

### 🛒 Модуль 3: Глобальний стейт (Cart Context)

- **Крок 10:** Створення `CartContext` для управління кошиком по всьому додатку.
- **Крок 11:** Реалізація функцій `addToCart` (додавання товару з початковою кількістю 1).
- **Крок 12:** Додавання логіки зміни кількості: `incrementQuantity` та `decrementQuantity`.
- **Крок 13:** Створення механізму видалення товарів з кошика та перевірки `isInCart`.
- **Крок 14:** Інтеграція `localStorage` у `useEffect` для збереження стану кошика після перезавантаження сторінки.

### 🗺 Модуль 4: Навігація та Роутинг

- **Крок 15:** Встановлення `react-router-dom` та налаштування `BrowserRouter` в `main.jsx`.
- **Крок 16:** Створення компонента `AllRoutes` для визначення шляхів `/` (Catalog) та `/cart` (Cart).
- **Крок 17:** Створення компонента `Header` з навігаційними посиланнями `Link` та лічильником товарів.
- **Крок 18:** Реалізація `CartPage`: відображення обраних товарів, керування їх кількістю та підрахунок загальної суми замовлення.
- **Крок 19:** Створення сторінки `ProductDetailsPage` (`/product/:id`) з відображенням ID товару.
- **Крок 20:** Наповнення `ProductDetailsPage` реальними даними знайденого товару (картинка, опис, ціна).
- **Крок 21:** Оновлення `products.js`:
  - Додавання ієрархічної структури: поля `group` (Батьківська категорія) та `subCategory`.
  - Заміна текстових заглушок зображень (`фото очікується`) на реальні посилання.

## 💡 Нотатки для навчання (Learning Notes)

- **Nested Objects**: Робота зі складними масивами (групування товарів за категоріями).
- **Sliding Intervals**: Як правильно очищувати `setInterval`, щоб не було витоку пам'яті.
- **URL Parameters**: Поглиблене вивчення `useParams` та `useLocation`.
